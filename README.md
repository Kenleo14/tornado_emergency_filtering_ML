# tornado_emergency_filtering_ML
Sub-Topic/Title: Deep Learning for Social Media Tornado Emergency Filtering 
Team Members: Kenneth Fulton Dataset Crisis NLP https://crisisnlp.qcri.org/data/iscram2013_nuggets_datasset/ISCRAM2013_dataset.zip Description: Human-labeled tweets collected during the 2011 Joplin Missouri tornado

To train the model: 

1. Clone repo @https://github.com/Kenleo14/tornado_emergency_filtering_ML.git

2. Download dependencies using requirements.txt

3. (optional) Run preprocessing.py for cleaning up the dataset from scratch. Otherwise, running train_model.py will work as long as "processed_joplin_binary.csv" is present.

4. Execute train_model.py
